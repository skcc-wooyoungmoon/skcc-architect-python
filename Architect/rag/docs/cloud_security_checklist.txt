# 클라우드 보안 필수 체크리스트

1.  ID 및 접근 관리 (IAM): 최소 권한의 원칙에 따라 사용자 및 서비스 계정에 역할을 할당합니다. MFA(다단계 인증)를 필수로 적용해야 합니다.
2.  네트워크 보안: Virtual Private Cloud (VPC)를 사용하여 네트워크를 격리하고, 보안 그룹(Security Groups)과 네트워크 ACL을 통해 인바운드/아웃바운드 트래픽을 엄격하게 제어합니다. Public IP 노출을 최소화합니다.
3.  데이터 암호화: 전송 중 데이터(TLS/SSL)와 저장된 데이터(서버 측 암호화)를 모두 암호화해야 합니다. 고객 관리형 키(CMK) 사용을 고려합니다.
4.  로깅 및 모니터링: 모든 API 호출, 시스템 로그, 네트워크 트래픽 로그를 수집하고 정기적으로 감사합니다. 이상 징후 탐지 시스템을 구축합니다.
5.  규정 준수: 서비스가 GDPR, HIPAA, ISMS 등 특정 규정을 준수해야 하는지 확인하고 관련 보안 통제를 적용합니다.
